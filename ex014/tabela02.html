Tabelas Semanticas

Como é a tabela semantica

Para tabela semantica usamos a anatomia:

  • TABLE

    • THEAD
      • TR, TH, TD

    • TBODY
      • TR, TH, TD

    • TFOOT
      • TR, TH, TD


Para deixar com efeito zebrado usamos o pseudo classe chamado :nth-child(2n)

Exemplo
<style>
tbody > tr:nth-child(2n) {
background-colo: #625272;
}
</style>



Temos que colocar o :nth-child(2n) no tr que é a linha da tabela, e o (2n) do :nth-child(2n) é que vai chamar todo numero par para zebrar se quiser impar tem que ser :nth-child(2n-1)
