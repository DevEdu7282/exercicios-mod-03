Scope na Tabela

Na tabela toda vez que usamos th usamos na teg scope que significa em qual direção o th está se referindo

Os valores do scope são:

  • Row para Direita
  • Rowgroup para conjuntos que estão a Direita
  • Col para Baixo
  • Colgroup para conjuntos que estão para Baixo

Caso esteja perdido jeva o Codigo no VS
