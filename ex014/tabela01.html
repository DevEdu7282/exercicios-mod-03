Abrendendo Tabela


Herarquia de Tabela simples

Na Herarquia de Tabela temos o table = TABELA e temos o table row = LINHA DE TABELA que fica por dentro da table e o table header = CABEÇALHO DE TABELA e por ultimo o table tada = DADO DE TABELA.

  • table = TABELA (table)
    • table row = LINHA DE TABELA (tr)
      • table header = CABEÇALHO DE TABELA (th)
      • table tada = DADO DE TABELA (td)
