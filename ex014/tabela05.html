Como usar o "colgroup"

Usamos o colgroup dentro da tag table para colocarmos uma cor na coluna da tabela, para n√£o temos que ficar colocando class em cada uma delas

Dentro do colgroup colocamos col que representa cada coluna na tabela e acresentamos a uma class a ela

  • Exemplo

    <table>
    ...<colgroup>
    .....<col class="amarelo">
    .....<col class="red" span="2">
    .....<col class="amarelo">
    ...</colgroup>
    </table>
