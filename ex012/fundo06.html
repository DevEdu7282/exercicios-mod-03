Fundo Fixo de tela

Como deixar o fundo fixo ou com scroll

Para deixar o fundo fixo usamos o valor fixed no background-attachment: fixed; para apenas o conteudo se mecher e não a imagem.

Para deixar o fundo com Scrll temos que usar o background-attachment: scroll; que por padrão já vai está
